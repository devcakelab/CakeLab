================================================================================
RUNTIME ENTRY FLOW (WINDOWS)
================================================================================

1) `start_system.bat`
   - Launches backend and frontend in separate terminals.
   - Opens `http://localhost:5173`.
2) `start_backend.bat`
   - Runs Django API on `0.0.0.0:8000`.
3) `start_frontend.bat`
   - Installs frontend dependencies if missing, then runs Vite dev server.

Default URLs
- Frontend: `http://localhost:5173`
- Backend API: `http://127.0.0.1:8000/api`

================================================================================
INTEGRATION SUMMARY
================================================================================

Frontend API pattern
- Frontend uses `frontend/src/lib/api.js`.
- Hostname is derived from current browser URL.
- Session auth uses cookies (`withCredentials: true`).

Backend API mount
- All app routes are under `/api/` via `backend/config/urls.py` -> `backend/core/urls.py`.

Reporting drill-down path
- Summary rows: `/api/reports`
- Per-period details: `/api/reports/details`
